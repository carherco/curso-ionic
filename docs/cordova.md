# APIs de Cordova

<https://ionicframework.com/docs/native/>

## Cómo incorporar un plugin de cordova

1) Instalación

> ionic cordova plugin add cordova-plugin-camera
> npm install --save @ionic-native/camera

2) Añadir el plugin al módulo raíz (app.module)
